# Simple Calculator

A simple calculator web app built with HTML, JavaScript, and Tailwind CSS.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division
- Percentage and decimal support
- Clear and delete functions
- Responsive design using Tailwind CSS

## Getting Started

### Using Tailwind CDN

Just open `cal.html` in your browser.

### Using Tailwind via npm (recommended for development)

1. **Install dependencies:**
   ```sh
   npm install
   ```

2. **Build Tailwind CSS:**
   ```sh
   npm run build:css
   ```

3. **Open `cal.html` in your browser.**
